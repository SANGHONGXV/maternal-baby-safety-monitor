# 母婴产品安全监测 Skill

一个专门用于监测近7天母婴育儿类产品安全事件的智能工具，生成适配母婴自媒体公众号风格的风险警示报告。

## 功能概述

本 Skill 帮助用户：
- 监测近7天内母婴产品的安全事件
- 自动整合多来源信息（官方通报、权威媒体、用户投诉、社交媒体）
- 生成专业的风险警示报告
- 提供可执行的避坑建议

## 监测品类

| 分类 | 品类 |
|------|------|
| **食品类** | 奶粉、辅食 |
| **用品类** | 奶瓶、纸尿裤、婴儿车、服装 |
| **家电类** | 母婴家电用品（温奶器、消毒柜、婴儿监护仪等） |

## 触发方式

使用以下关键词触发：

```
母婴安全监测、产品安全事件、婴儿用品风险、奶粉安全、
辅食安全、母婴产品召回、育儿用品警示、宝宝用品安全、
婴幼儿产品问题
```

## 使用方法

1. 触发 Skill 后，系统会自动搜索近7天的母婴产品安全信息
2. 按照信息来源优先级整合数据
3. 生成 Markdown 格式的风险警示报告

## 报告输出

- **默认输出路径**：`e:/16每周分享/公众号自动化/母婴产品安全监测_{日期}.md`
- **文件名格式**：`母婴产品安全监测_YYYY-MM-DD.md`

## 报告结构

每份报告包含：

1. **标题**：养娃避坑！近7天母婴产品安全预警
2. **导语**：吸引注意力的开篇
3. **分类警示**：
   - 食品类警示（奶粉、辅食）
   - 用品类警示（奶瓶、纸尿裤、婴儿车、服装）
   - 家电类警示
4. **事件详情**（每起事件包含6个部分）：
   - 事件品牌
   - 事件日期
   - 事件详情
   - 核心风险提示
   - 最新处置进展
   - 信息来源
5. **宝妈实操建议**：可执行的避坑措施
6. **结尾互动**：引导用户评论互动

## 信息来源优先级

1. **官方通报**（最高优先级）：国家市场监管总局、各省市监局、海关总署
2. **权威媒体报道**：新华社、人民日报、央视新闻、南方都市报
3. **用户投诉平台**：黑猫投诉、消费保
4. **社交媒体热议**：微博、小红书（需有具体产品/品牌指认）

## 风格特点

- 口吻亲切，像闺蜜/姐妹聊天
- 专业易懂，避免晦涩术语
- 关键信息用**粗体**突出
- 所有信息标注来源，确保真实可信

## 更新说明

- 报告每周定期生成
- 仅收录近7天内的新事件
- 持续跟踪已有事件的最新进展

> ⚠️ 本 Skill 仅供信息参考，不构成法律建议。具体产品问题请以官方通报为准。
