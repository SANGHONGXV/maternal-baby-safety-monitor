# 母婴产品安全监测 Skill

## 目的

监测近7天母婴育儿类产品安全事件，生成适配母婴自媒体公众号风格的风险警示报告，帮助宝妈快速获取关键信息并采取应对措施。

## 监测品类（按优先级排序）

| 分类 | 品类 |
|------|------|
| **食品类** | 奶粉、辅食 |
| **用品类** | 奶瓶、纸尿裤、婴儿车、服装 |
| **家电类** | 母婴家电用品（温奶器、消毒柜、婴儿监护仪等） |

## 信息来源优先级

1. **官方通报**（最高优先级）：国家市场监管总局、各省市监局、海关总署
2. **权威媒体报道**：新华社、人民日报、央视新闻、南方都市报
3. **用户投诉平台**：黑猫投诉、消费保
4. **社交媒体热议**：微博、小红书（需有具体产品/品牌指认）

## 搜索策略

### 必搜关键词组合

```
近7天日期范围 + 品类关键词 + 安全/召回/问题/曝光

示例：
- "2026年4月 奶粉 召回/问题/曝光/安全"
- "2026年4月 纸尿裤 细菌/污染/质量问题"
- "2026年4月 婴儿车 缺陷/召回/安全隐患"
```

### 搜索步骤

1. **第一轮**：搜索官方通报（国家市场监管总局官网、总局公告）
2. **第二轮**：搜索权威媒体报道（用 `site:news.sina.com.cn OR site:news.163.com` 限制权威媒体）
3. **第三轮**：搜索投诉平台（黑猫投诉官网）
4. **第四轮**：补充社交媒体（如前几轮信息不足时）

## 事件结构模板

每起事件必须包含以下6个部分，缺一不可：

### 【事件品牌】
明确品牌全称，无模糊表述（如"某知名品牌"不可用，应写"爱他美"或"美赞臣"）

### 【事件日期】
精确到具体日期YYYY-MM-DD，多日期按时间线标注

### 【事件详情】
- **发现过程**：用户投诉/官方抽检/媒体曝光/企业自检
- **问题描述**：核心问题是什么
- **涉及产品型号/批次**：精准标注，无批次则说明"批次待查"
- **影响范围**：涉及国家/地区、中国市场是否波及、影响人群数量

### 【核心风险提示】
- **健康危害**：具体危害（如中毒、烫伤、肠道感染、窒息风险等）
- **安全隐患**：结构缺陷、污染、残留等
- **适用人群风险差异**：**重点说明**婴幼儿与成人/其他人群的风险不同
  - 婴幼儿体重轻，单位体重摄入/接触量更高
  - 婴幼儿代谢系统未发育完善，排毒能力弱
  - 婴幼儿免疫力低，感染风险更高
  - 婴幼儿无自我保护意识，无法主动规避风险

### 【最新处置进展】
近7天内最新动态：
- 官方举措（立案调查、下架、整改通知）
- 品牌回应（声明、致歉、赔偿方案）
- 召回情况（召回范围、召回渠道）
- 维权进展（退款、退货、赔偿）

### 【信息来源】
标注来源名称和URL，增强可信度

## 无事件品类的处理

如某品类近7天无符合条件的事件，标注：

> **【XXX品类】**
> 近7天暂未监测到明确官方通报、权威报道及用户投诉的相关问题事件，后续持续关注。

## 报告结构

### 标题格式
`养娃避坑！近7天母婴产品安全预警（{日期}）`

### 导语（吸引宝妈）
```
⚠️ 养娃无小事！过去7天，又有哪些母婴产品被曝出安全问题？
事关宝宝健康，@所有宝妈奶爸，速看避坑！
```

### 正文分类结构

```
一、食品类警示（奶粉、辅食）
   └─ 事件1
   └─ 事件2（如有）
   
二、用品类警示（奶瓶、纸尿裤、婴儿车、服装）
   └─ 事件1
   └─ 事件2（如有）
   
三、家电类警示
   └─ 事件1（如有）
```

### 结尾互动引导
```
📢 你家在用这些产品吗？评论区聊聊你的使用体验或避坑经验！
   
💡 关注我，不错过每一期安全预警！
```

### 宝妈实操建议（结尾固定模块）
```
🛡️ 宝妈实操建议：
1. 立即检查家中是否有涉及品牌/批次的产品
2. 如有疑虑，可拨打品牌官方客服核实
3. 保留购买凭证，以便后续维权
4. 关注「国家市场监管总局」公众号，获取第一手信息
```

## 风格要求

| 要求 | 说明 |
|------|------|
| **口吻** | 亲切接地气，像闺蜜/姐妹聊天 |
| **专业度** | 通俗易懂，避免专业晦涩术语 |
| **重点突出** | 关键风险点、涉及品牌用**粗体** |
| **信息真实** | 所有信息需标注来源，不可编造 |
| **时效性** | 仅收录近7天内事件 |
| **地域** | 重点突出中国市场影响 |
| **互动性** | 引导评论、收藏、转发 |
